##### Companion to Chapter 4 of the textbook [Machine Learning](http://www.cs.cmu.edu/~tom/mlbook.html).

A neural network learning algorithm called Backpropagation is among the most effective approaches to machine learning when the data includes complex sensory input such as images. This web page provides an implementation of the Backpropagation algorithm described in Chapter 4 of the textbook [Machine Learning](http://www.cs.cmu.edu/~tom/mlbook.html). It also includes the dataset discussed in Section 4.7 of the book, containing over 600 face images.

### Documentation

This documentation is in the form of a homework assignment (available in [postscript](http://www.cs.cmu.edu/afs/cs.cmu.edu/project/theo-8/faceimages/docs/hw97.ps) or [latex](http://www.cs.cmu.edu/afs/cs.cmu.edu/project/theo-8/faceimages/docs/hw97.tex) ) that provides a step-by-step introduction to the code and data, and simple instructions on how to run it
